🔐 Authentication

Use your secret API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY

📡 Sample Request

POST https://dikanna-voice-api.onrender.com/voice

{
  "text": "Ndewo",
  "model_id": "eleven_multilingual_v2"
}

Returns an MP3 stream with accurate tonal pronunciation.
